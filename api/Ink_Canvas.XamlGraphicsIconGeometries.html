Remarks

该类定义了应用程序中使用的所有工具图标，包括：

  • 光标图标
  • 画笔图标
  • 橡皮擦图标
  • 选择工具图标
  • 手势图标
  • 老版本浮动栏图标 每个图标都有实线(Lined)和虚线(Solid)两种变体
